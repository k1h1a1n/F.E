End-to-End Encryption:
Durity uses end-to-end encryption to guarantee the security of your files in the cloud.
Using this technology, Durity encrypts every file and relevant file metadata on your devices with the unique, encryption key. These keys are never sent to our servers in an unencrypted format.

Unlike other services, Durity never transmits or stores files, encryption keys and user passwords in unencrypted or unhashed form.
